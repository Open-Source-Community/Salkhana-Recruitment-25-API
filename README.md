# Salkhana-Recruitment-25-API
Salkhana-Recruitment-25-API streamlines the backend committee's interview process by allowing interviewers to define questions based on set criteria. It also tracks interviews by locally storing interviewer and interviewee names, ensuring an organized and efficient recruitment workflow.

# how to run

1. clone the project
2. open it on your computer
3. run 
    * ```npm install```
    * ```npm install nodemon```
    * ```npm run dev```
4. after the server starts open the index.html file
